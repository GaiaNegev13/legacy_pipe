# Legacy Pipe Project
This project was created as part of my master thesis at Yaniv Assaf's Lab (YaLab).
The thesis subject is Brain Structure Across Generations - Changes in Grey Matter Volume Along Birth Years 

## Project Goals
1. Preprocess scans that were aquired since 2000, from raw dicom files to processed, BIDS organized nii files. 
2. Extract biological and technical properties for each scan
3. Statistical analysis and validation of the data
